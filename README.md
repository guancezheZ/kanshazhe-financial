# 观测者企业财务模拟系统（教学用）

> 基于 Vue3 + Element Plus + Vite 的财务模拟教学系统，已集成 Tauri v2 桌面应用。

## 功能概览

| 模块 | 说明 |
|:-----|:------|
| 📒 **凭证管理** | 录入/查询/审核/签字/过账，支持教学自动过账 |
| 📊 **账簿报表** | 科目余额表/试算平衡表/三大报表/自定义报表 |
| 💰 **出纳管理** | 现金/银行日记账、银行对账（自动勾对） |
| 🏢 **资产管理** | 固定资产卡片、折旧（直线法/双倍余额法/年数总和法） |
| 📋 **辅助核算** | 部门/员工/客户/供应商/项目 5种核算维度 |
| 📝 **凭证模板** | 模板CRUD，一键应用跳转录入 |
| 🔄 **外币核算** | 8种币种汇率，一键获取最新汇率 |
| 📑 **期末处理** | 自动结转损益、结账/反结账 |
| 🧾 **模拟纳税申报** | 增值税+企业所得税自动计算提交 |
| 🔍 **审计轨迹** | 操作日志自动记录 |

## 🎓 教学系统

四行业共 **2,122 个教学任务** + **9 个案例库**：

| 行业 | 任务数 | 特点 |
|:----|:------:|:-----|
| 🏭 制造业 | 586 | 成本核算、约当产量法、制造费用分配 |
| 🏪 商业企业 | 531 | 进销存、增值税发票、促销折扣 |
| 💼 服务业 | 504 | 项目核算、人工成本、完工百分比 |
| 🏗️ 建筑业 | 501 | 合同成本、分包核算、材料管理 |

### 教学模式
- **分步引导**：Step-by-Step 推理 → 自动填入
- **练习模式**：填空作答 → 自动比对 → 错误讲解
- **考试模式**：独立作答 → 无提示 → 评分

### XP/等级/成就
- 13 级声望称号（见习生→财务传说）
- 16 项成就徽章
- 场景完成成就各 500 XP

## 🛡️ 安全体系

- **激活码验证**：16位 XOR 校验和 + Cloudflare Worker 在线验证 + 离线回退
- **单设备绑定**：同一码绑定 1 台设备（固定指纹，平台版专用）
- **设备指纹**：确定性算法（去随机化），清缓存后不变
- **localStorage 加密**：AES-256-GCM 双轨加密（同步明文+异步密文）
- **完整性校验**：SHA-256 数据模块哈希比对
- **Worker 防护**：速率限制 + IP 封禁（20次失败/小时）
- **Rust 加固**：硬件指纹 + 密钥拉伸 100 轮迭代

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器（端口 3000）
npm run dev

# 运行全部测试（455项）
npm run test

# Playwright 全量自动化测试（56项）
BASE_URL=http://localhost:3000 node scripts/run-all-tests.cjs

# 构建 Tauri 桌面应用
export PATH="$HOME/.cargo/bin:$PATH" && npx tauri build
```

### 系统要求
- **Node.js**: v24.16.0+
- **npm**: 11.13.0+
- **Rust** (Tauri 构建): 最新 stable
