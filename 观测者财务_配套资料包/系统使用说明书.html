1. 第一次使用——登录和准备工作

1.1 打开系统

启动方式有两种：

方式操作
一键启动在项目文件夹里双击 start.bat → 等待黑窗口跑完 → 浏览器自动打开
手动启动打开终端/命令提示符 → 输入 npm run dev → 浏览器访问 http://localhost:3000
关闭系统双击 stop.bat 或按 Ctrl+C 关闭终端

1.2 登录系统

你会看到什么：一个居中显示的大白框，上面写着"観測者企业财务模拟系统"，中间有用户名和密码输入框。
📌 上图：登录页面——试用账号 admin / admin123
1 输入用户名
点击"用户名"输入框（有一个小人物图标），输入 admin
2 输入密码
点击"密码"输入框（有小锁图标），输入 admin123（密码不会明文显示，会用圆点代替）
3 点击登录
点击蓝色"登 录"按钮（也可以直接按键盘上的 Enter 键）
✅ 登录成功！你会自动跳转到"工作台"页面。
💡 如果登录失败？检查用户名密码是否输对了，注意大小写。系统目前是模拟登录，不需要注册账号。

1.3 激活系统——首次使用必须做

首次登录后，系统会弹出激活窗口。 本系统需要激活码才能使用全部功能。如果你还没有激活码，请联系管理员获取。
1 获取激活码
从管理员处获取 16 位激活码，格式如 AB12-CD34-EF56-7890
2 输入激活码
在弹出的激活窗口中，输入你的 16 位激活码 → 点击"激活"按钮。
  • 在线激活（推荐）：系统会自动联网验证，激活码绑定你的设备
  • 离线激活：如果无法联网，系统会自动使用本地验证（同一激活码绑定 1 台设备）
✅ 激活成功！弹出"激活成功"提示，工作台数据加载完成。
💡 怎么知道激活了没有？ 进入"系统管理 → 激活管理"，可以查看当前激活状态和已绑定的设备信息。如果更换电脑，需要联系管理员重新激活。

1.4 登录后第一件事——看看工作台

你会看到什么：左上角有"工作台"三个字，下面是几个快捷按钮（新增凭证、科目表等），再往下是四张统计卡片（本月凭证数、借方合计、贷方合计、试算平衡）。
📌 上图：工作台——快速入口 + 财务数据概览卡片

工作台上的信息告诉你：

  • 快捷操作：五个大按钮——点他们可以直接跳到最常用的功能
  • 本月凭证数：这个月一共做了多少张凭证
  • 借方合计/贷方合计：所有金额加起来，借和贷应该相等
  • 试算平衡：绿色✓表示账是平的，红色✗表示有问题
  • 当前账套：你正在使用的账套信息（名称、年度、本位币等）

1.5 切换角色

这个系统有三种角色——会计、出纳、主管。不同的角色能做的事情不一样。登录后默认是"会计"。

1 找到角色标签
看页面右上角，有一个蓝色的小标签，上面写着"会计"或"👤 会计"——这就是你的当前角色
2 点击切换
点击"会计"标签 → 弹出一个下拉菜单 → 里面有"会计""出纳""主管"三个选项
3 选择角色
点击你想切换的角色（比如"出纳"）→ 页面会自动刷新 → 左上角菜单会变成对应角色能用的功能
✅ 切换成功！比如切到"出纳"后，左侧菜单的"出纳管理"就能用了。
角色主要能做什么
👤 会计做凭证、查账簿、出报表、管理科目、成本核算——日常做账的主力
💰 出纳登记现金日记账和银行日记账、银行对账、给凭证签字——管钱的
👑 主管审核凭证、管理期间、审计追踪——管人的
💡 新手提示：如果你是第一次接触财务软件，建议先用"会计"角色，因为大部分核心功能都在这。

2. 科目管理——会计的"文件夹"

路径：左侧菜单栏 → 总账 → 科目表

科目是什么？ 就像电脑里的文件夹——把类似的经济业务放到一个类别里。系统已经预置了 60 多个常用科目，你也可以自己添加。

2.1 认识科目表界面

你会看到什么：页面分成左右两半——左边是科目树（像文件夹目录一样展开），右边是科目详情（点击左边某个科目，右边显示它的详细信息）。
📌 上图：科目表——左树右详，60+预置科目

2.2 查看一个科目

1 在左侧科目树中找到你感兴趣的科目
比如找到"银行存款(1002)"——前面有个小三角▶，点击可以展开看下面的子科目（工商银行、建设银行）
2 点击科目名
点击"银行存款" → 右侧会显示该科目的详细信息（编码、名称、类型、余额方向等）

2.3 新增一个科目

比如公司新开了"招商银行"账户，需要在"银行存款"下新增一个子科目：

1 点击父科目
在左侧科目树中点击"银行存款(1002)"（选中它，背景变蓝）
2 点击"新增下级"按钮
在页面右上角找到并点击"新增下级"按钮 → 右边会弹出一个对话框
3 填写科目信息并保存
在弹出的表单中填写：科目名称 = "招商银行"，编码可以留空（系统自动生成）。点击"确定"保存。
✅ 新科目"银行存款→招商银行(100203)"就创建好了！你可以在科目树中展开银行存款看到它。
💡 删除保护：如果一个科目已经被凭证使用过了，或者下面还有子科目，系统会禁止删除——这是为了防止数据错误。想删除的话，得先把相关的凭证删掉。

3. 凭证管理——做账的核心工作

凭证是财务核算的基础——每一笔业务都要先做成凭证，然后数据才能流入账簿和报表。

3.1 新增一张凭证（完整步骤演示）

路径：左侧菜单 → 总账 → 凭证录入

你会看到什么：一个空白的凭证表格——上面是"凭证头"（凭证号、日期、附件数），下面是"分录表格"（摘要、科目、借方金额、贷方金额）。
📌 上图：凭证录入页面——分录表格 + 科目树选择 + 借贷自动校验

我们来演示一个实际案例：公司用工商银行账户支付办公用品费 500 元。

1 填写日期
点击"日期"输入框 → 选择今天的日期（或业务实际发生的日期）
2 填写第一行分录（借方——费用增加）
  • 摘要：输入"购买办公用品"（描述这笔业务是什么）
  • 科目：点击科目列 → 弹出一个科目选择窗口 → 在上面搜索框打"管理费用"或编码"6602" → 找到"管理费用─办公费(660201)" → 点击选中
  • 借方金额：输入 500
  • 贷方金额：留空（同一行不能同时填借和贷）
3 填写第二行分录（贷方——银行存款减少）
  • 摘要：输入"购买办公用品"（同一个业务摘要可以相同）
  • 科目：搜索"工商银行"或"100201" → 选中"银行存款─工商银行(100201)"
  • 借方金额：留空
  • 贷方金额：输入 500
4 检查借贷是否平衡
看表格下方——系统会自动计算合计。如果显示✓ 借贷平衡，就说明没问题。
如果显示✗ 不平衡，检查金额是否填对了（借方合计必须 = 贷方合计）。
5 保存凭证
点击左上角的"保存"按钮 → 系统提示"保存成功"。
✅ 凭证已保存！状态为"草稿"——接下来要经过签字→审核→过账流程。

3.2 如何选择会计科目（一个详细的教程）

点开"科目"列时，你会看到一个科目选择窗口，长这样：

┌─ 选择科目 ────────────────────────┐ │ [输入科目编码或名称搜索...] │ │ │ │ ├ 1001 库存现金 │ │ ├ 1002 银行存款 │ │ │ ├ 01 工商银行 │ │ │ ├ 02 建设银行 │ │ │ └ 03 招商银行 │ │ ├ 1122 应收账款 │ │ │ ├ 01 甲公司 │ │ │ └ 02 乙公司 │ │ ... │ └────────────────────────────────────┘

使用技巧：可以直接输入编码快速定位——比如输入 100201 就会直接跳到"工商银行"。

3.3 凭证的四种状态——理解凭证的"生命旅程"

状态含义能做什么
📄 草稿 (DRAFT)刚创建，还没人检查过可修改、可删除。出纳可以签字，主管可以审核
✍️ 已签字 (SIGNED)出纳确认了（涉及现金/银行的凭证）主管可以审核。出纳可以取消签字
✅ 已审核 (APPROVED)主管复核通过了可以过账。主管可以取消审核
📊 已过账 (POSTED)数据已进入账簿和报表不可修改。可以先反过账，再修改

3.4 签字、审核、过账操作

路径：左侧菜单 → 总账 → 凭证查询

你会看到什么：上面有一排筛选条件（期间、日期、状态、关键词），下面是凭证列表表格。列表中的每一行代表一张凭证。

操作一：查询凭证

1 设置筛选条件
  • 期间：输入 YYYYMM 格式，比如 202606（表示 2026 年 6 月）
  • 日期范围：可以选开始和结束日期
  • 状态：选"草稿""已审核"或空（全部）
  • 关键词：输入摘要里的文字或科目名称
2 点击查询
点击蓝色"查询"按钮 → 下面表格会显示符合条件的凭证

操作二：出纳签字

需要角色：登录后右上角角色标签必须是"出纳"（不会切换？看 1.4 节）
适用场景：凭证里有"库存现金"或"银行存款"科目时
1 找到草稿状态的凭证
在凭证列表中找到状态为"草稿"的凭证（看表格中的"状态"列）
2 点击"签字"
选中该行 → 点击凭证列表上方的"签字"按钮（如果没有看到，确认你的角色是"出纳"）
✅ 状态变为"已签字"！如果签错了，选中后点"取消签字"可以撤回。

操作三：主管审核

需要角色：必须是"主管"
1 切换到主管角色
点击右上角角色标签 → 切换为"主管"
2 找到已签字的凭证
在凭证列表中筛选状态为"已签字"的凭证
3 点击"审核"
选中该凭证 → 点击"审核"按钮
✅ 状态变为"已审核"！反审核可以取消。

操作四：过账

需要角色：会计或主管
1 找到已审核的凭证
筛选状态为"已审核"的凭证
2 点击"过账"
选中后点击"过账" → 系统提示确认 → 确定
✅ 过账成功！数据已经进入科目余额表和报表中了。如果发现错了，可以"反过账"修改。
💡 教学模式下的区别：如果你是在教学模式下做任务（答对了），系统会自动跳过签字和审核，直接过账。这是为了方便学习，不是标准的真实流程。

3.5 凭证查询——筛选与批量操作

凭证查询页不仅能筛选查看凭证，还支持批量签字/审核/过账：

📌 上图：凭证查询——筛选条件 + 凭证列表 + 操作按钮
1 设置筛选条件
期间（YYYYMM格式）、日期范围、状态（草稿/已签字/已审核/已过账）、关键词搜索

点击列表中某一行 → 该凭证高亮选中。按住 Ctrl 可以多选，批量操作

3.6 凭证模板——快速录入常用业务

路径：总账 → 凭证模板

对于每月都会发生的固定业务（如计提折旧、结转工资），可以使用凭证模板——一次设置，每月调用。

📌 上图：凭证模板——预设常用分录，一键生成凭证

在模板列表中找到需要的模板 → 点击"生成凭证" → 系统自动填充分录到新凭证

在凭证查询列表中，点击某张凭证的"附件"图标或双击凭证行，可以查看该凭证的完整信息，包括原始单据影像：

📌 上图：凭证详情弹窗——显示完整分录 + 原始单据附件

教学模式下，每道题都配有对应的原始凭证（发票、回单、收据、支票等），点击浮动窗的"附件"按钮即可查看——和真实单据一模一样，帮助你建立"凭证←→原始单据"的对应关系。

4. 期末处理——月底结账

4.1 期末结转（把费用和收入转到"本年利润"）

路径：左侧菜单 → 报表 → 期末结转

为什么要做这一步？ 每个月月底，要把这个月的收入和费用汇总，算出来这个月赚了还是亏了。这个"汇总"的过程就叫"结转"。

1 选择期间
确认左上角的期间选择器显示的是你要结转的月份（比如 202606 就是 2026 年 6 月）
2 查看收支列表
页面上会显示两个表格——收入类科目和费用类科目。每个表格里列出这个月有发生额的科目和金额。看看对不对。
3 执行结转
点击右上角的"执行结转"按钮 → 系统提示确认 → 确定
✅ 结转成功！系统会自动生成一张结转凭证（把收入转到本年利润贷方，费用转到本年利润借方）。
📌 上图：期末结转——收入类科目和费用类科目汇总
⚠️ 注意：结转前要先确保这个月所有凭证都已经过账了。每个月只能结转一次。

4.2 结账/反结账

路径：左侧菜单 → 系统管理 → 会计期间

什么是结账？ 确认这个月的账已经做完、不再修改了→"关闭"这个月。关闭后这个月的凭证不能再改。

📌 上图：会计期间管理——全年12个月的结账状态

什么是结账？ 确认这个月的账已经做完、不再修改了→"关闭"这个月。关闭后这个月的凭证不能再改。

1 查看期间列表
页面上会列出所有月份的期间——"2026年01月"到"2026年12月"，每个行有"状态"列（开启/已结账）和"操作"列（结账/反结账）
2 点击"结账"或"反结账"
  • 如果状态是"开启" → 点"结账" → 确认 → 该月关闭
  • 如果状态是"已结账" → 点"反结账" → 确认 → 重新打开
✅ 操作成功！注意：结账一般在所有凭证过账+期末结转之后再做。

5. 查看报表

三种报表都在：左侧菜单 → 报表 → 选你要看的表

报表看什么的
资产负债表公司到这一天为止有多少家底（资产多少、欠债多少、股东权益多少）
利润表公司这段时间赚了多少、花了多少、净赚多少
现金流量表公司这段时间钱从哪来、到哪去了（经营/投资/筹资）

5.1 查看资产负债表

1 选择期间
进入页面后，上方（通常是日期选择器）选择你要查看的月份，比如 2026 年 6 月
2 查看数据
页面自动显示资产负债表，左边是"资产"（流动资产+非流动资产），右边是"负债"和"所有者权益"。最下方应该满足资产 = 负债 + 所有者权益
📌 上图：资产负债表——资产 = 负债 + 所有者权益

5.2 查看利润表

1 选择期间
选择要查看的月份
2 查看数据
页面从"营业收入"开始，逐行减去各项成本和费用，最后得出"净利润"——如果净利润是正数表示赚钱了，负数是亏了
📌 上图：利润表——从营业收入到净利润
📌 上图：现金流量表——经营活动/投资活动/筹资活动

8. 数据安全与完整性保护

财务数据是公司最重要的资产之一。本系统在安全方面做了多重保护，确保你的数据安全可靠。

8.1 数据存储在哪里

所有数据存储在浏览器本地（localStorage），不会上传到任何云端服务器。这意味着：
  • ✅ 你的财务数据始终在你自己的电脑上
  • ✅ 不需要联网也能使用（首次激活除外）
  • ⚠️ 如果清除浏览器缓存，数据会丢失——建议定期备份

8.2 数据加密保护

系统对敏感数据进行了双轨加密保护：

  • 明文同步写入：确保读取速度最快
  • AES-GCM 异步加密：同时保存一份加密版本（2 秒防抖），防止数据被篡改
  • 加密范围包括：激活码、设备指纹、场景数据、XP 数据等所有敏感信息

8.3 激活码安全机制

  • 激活码为 16 位大写 hex 格式（例如 AB12-CD34-EF56-7890）
  • 同一激活码绑定 1 台设备（设备指纹固定，更换电脑需重新激活）
  • 激活验证首选在线验证，Worker 不可达时自动回退到本地校验
  • 连续 20 次激活失败会临时封禁 1 小时（防暴力破解）

8.4 数据完整性自检

系统启动时会自动检查 7 大核心数据模块的完整性：

  • 制造业教学数据（入门+2~12月）
  • 商业企业教学数据（12个月）
  • 服务业教学数据（12个月）
  • 建筑业教学数据（12个月）
  • 案例库数据（9个案例）

如果检测到数据被篡改或损坏，弹窗提醒你重新安装。这个检查完全静默运行，不影响正常使用。

💡 我的数据安全吗？ 完全安全。系统不联网收发数据（除激活验证外），没有后门，不走云端。即使有人物理接触到你的电脑，没有登录密码也无法查看加密内容。

10. 常见问题

❓ 保存凭证时提示"借贷不平衡"怎么办？
✅ 检查一下借方金额合计和贷方金额合计是否相等。常见错误：同一行里既填了借方又填了贷方，应该只填一方。

❓ 点了"保存"没有反应？
✅ 检查是否必填项没填（日期、摘要、科目都是必填的）。另外分录至少要有两行（一借一贷）。

❓ 修改不了已过账的凭证？
✅ 需要先"反过账"（在凭证查询中找到该凭证 → 点"反过账"）→ 然后就可以修改了 → 改完重新过账。

❓ 为什么看不到"出纳管理"这个菜单？
✅ 因为当前角色不是出纳。看页面右上角→点角色标签→切换到"出纳"即可。

❓ 资产负债表不平衡怎么办？
✅ 常见原因：① 损益类科目没有结转 ② 还有凭证是草稿状态没有过账 ③ 做凭证时借贷方向搞反了。先去检查这三个方面。

❓ 教学中想切换模式但不知道怎么操作？
✅ 打开浮动窗 → 顶部有"引导模式""练习模式""考试模式""自由练习"四个按钮 → 点击切换。切换后重新进入任务会按新模式处理。

❓ 怎么切换行业（制造业/商业/服务业/建筑业）？
✅ 在页面右上角找到行业名称标签（默认显示"制造业"）→ 点击 → 弹出行业列表 → 选择你想学的行业 → 系统自动加载该行业的教学数据。

❓ 之前做过的一些任务，数据还在吗？
✅ 数据保存在你浏览器的 localStorage 里（不会上传到任何服务器）。如果清除了浏览器缓存或开启了无痕模式，数据就没了。

❓ 想重新学一遍，怎么重置？
✅ 在教学中心页面 → 右上角找到"🔄 重置进度"按钮 → 点击确认 → 所有完成记录清空，账套也会重置到初始状态。

❓ 激活码丢了/激活失败怎么办？
✅ 如果激活码丢失，请联系管理员查询。激活失败时：① 检查网络是否通畅 ② 确认激活码格式正确（16位大写hex）③ 如果提示"设备已绑定"，说明该激活码已在一台设备上激活，更换电脑需要联系管理员重置。

❓ 案例库和教学任务有什么区别？
✅ 教学任务是按月份循序渐进的引导式学习；案例库是独立的真实企业账套，需要你独立完成从凭证到报表的全流程。建议先学完至少一个月教学，再挑战案例。

❓ 有没有系统的会计知识点教材？
✅ 有！系统赠送了完整的教学知识点文档（docs/教学知识点介绍.html），包含会计 12 章 + 出纳 8 章，共 65 节内容。支持搜索、书签、暗色模式，在软件目录的 docs/ 文件夹里可以打开。

❓ 报错信息看不懂怎么办？
✅ 可以截图发给我（Claude）看。常见的错误大多是因为科目没选对、金额填反了、或者借贷不平衡。
