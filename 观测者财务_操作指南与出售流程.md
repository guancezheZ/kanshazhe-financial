## 一、常用终端指令

```bash
npm run dev                    # 启动开发服务器（本地测试用）
npm run test                   # 跑全部测试（改代码后必须做！455项）
npx kill-port 3000             # 清理端口占用

# 生成与管理激活码
node scripts/key-manager.cjs gen 10          # 生成10个激活码
node scripts/key-manager.cjs export <令牌>    # 查看密钥状态
node scripts/key-manager.cjs upload <令牌>    # 上传密钥到Worker

# 打包桌面安装包
export PATH="$HOME/.cargo/bin:$PATH" && npx tauri build

# 提交代码到 GitHub
git add -A && git commit -m "说明" && git push
```

## 二、更新 Worker（Cloudflare）

1. 打开 https://dash.cloudflare.com
2. Workers → kanshazhe-activation → 代码
3. 用记事本打开 `scripts/worker-activation.js` → 全选复制
4. 替换 Cloudflare 编辑器里的代码 → Deploy

**发新版要改的地方：**
- Worker 第24-25行：蓝奏云链接和密码
- GitHub 创建 Release → 上传安装包
- 重新打包 Tauri（`npx tauri build`）
